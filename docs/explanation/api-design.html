post /login#

  • [x] mark user as online

  • [x] update operation time

  • [ ] register user’s current clients(issue client id)

  • [ ] issue client secret

  • Request

    • General:

      • Request Method: POST

    • Headers:

      • Content-Type: application/json

    • Payload:

      {
    "username": "user",
    "password": "xxxx"
}

  • Response

    • General：

      • Status Code: 200

    • Payload:

      • success:

        {
    "result": "succeeded",
    "client_id": "xxxxx",
    "client_secret": "xxxxx"
}

      • failure:

        {
    "result": "failed",
    "client_id": null,
    "client_secret": null
}
